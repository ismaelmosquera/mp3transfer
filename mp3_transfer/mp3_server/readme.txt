Mp3Server readme file

To run the server, execute the run.sh xcript if you are under a Linux like shell,
Or the run.bat file if you are under a MSWindows command line.
Since this is an application from the command line, you must go to this folder and
Execute the file as explained above.

Cd <path to this folder>

And execute the script according with your shell.
The first time you run the server, maybe your system will ask you for permission to
listening in port 11105, you must agree and accept in order to execute the program.
